Notes on 1.5
------------

- first: returns the first element of a list.
		there are similar ones like second, third ... up to tenth

- rest: returns the list minus the first element

-NOTE: first was called car, and rest cdr

- nth n: returns the nth elements of a list

- length: returns the length of a list

- cons object list: returns a new list with the object added to the front of the list

- append lists: takes any number of lists and joins them all together into a single list
